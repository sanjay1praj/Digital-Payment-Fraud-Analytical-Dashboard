# Digital Payment Fraud Analytical Dashboard

Interactive Power BI dashboard for monitoring digital payment fraud, fraud trends, and key risk indicators.

## Business Objective

Analyze digital payment transactions to identify fraud patterns, monitor fraud rates, and support data-driven decision making.

## Tools Used

- Power BI
- Excel
- DAX

## Key Features

- Fraud Rate Analysis
- Transaction Trend Analysis
- Payment Method Analysis
- Merchant Risk Analysis
- Device Analysis
- Geographic Fraud Analysis
- Interactive Filters and Slicers
